**Two Pointers** is a technique where we use **two indices (pointers)** to traverse a data structure (usually an array or string) instead of using nested loops.

## where to apply 2 pointers:

1. Array is **sorted**
2. Problem involves **pairs**
3. Need to **compare elements from both ends**
4. Need to **remove duplicates**
5. Need to **reverse something**
6. Sliding window type problems
7. Palindrome checking

Where not to use:
- unsorted 
- when random access is need na dont use this 

Directions:
1) Opposite Direction
2) Same Direction
3) Sliding Window Direction
4) Fast & Slow


PATTERN 1:
- 2 sum
- container with most water 
- 3 sum [retry]
- 3 sum closest [retry] 
- is-subsequence
- squares of an array
- boats to save people
