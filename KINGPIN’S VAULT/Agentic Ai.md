Ai Agents:

AGENTS VS AUTOMATION:
   Automation --->  pre-defined workflow in loop at particular time
   eg: automated plant watering system [The problem -- in edge cases like rainy season and summer season this wont work]
   
   Agents ---> work based on the real time situation like human brain and work correctly on edge cases
